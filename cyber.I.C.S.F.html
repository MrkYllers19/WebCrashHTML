7 JENIS SISTEM SERANGAN SIBER

1.PHISING=Yaitu,menyebarkan suatu link website jebakan atau link website palsu,tujuannya untuk mendapatkan informasi data dari korban,yang telah memberi/mengisi datanya lewat formulir atau login pada web phising palsu tadi.

Gambar1. Ilustrasi Phising

2.SCAM=Yaitu,Usaha untuk menipu korban secara langsung, guna mendapatkan informasi data dari korban,cara kerjanya adalah berpura-pura menjadi seorang sales online,lalu mengirimkan pesan/email ke korban, dengan iming-iming mendapat bonus/hadiah dari perusahaan resmi,contoh hadiah dari "shoopie,tokopedia,dll,lalu menyuruh korban untuk mengisi data formulir terlebih dahulu dan data yang telah di-isi lalu dikirim,langsung akan diketahui oleh pencuri data

Gambar2. Ilustrasi Scam

3.SOCIAL ENGINEERING=Yaitu,metode pencurian data dengan membuat sebuah iklan scam,dimana jika iklan tersebut diklick oleh korban, maka akan diarahkan ke sebuah situs website ,situs website ini dapat berisi malware berbahaya yang dapat menyusup ke sistem server perangkat kamu dan memata-matai,mencuri info data,serta dapat merusak sistem server perangkat anda.

Gambar3. Ilustrasi SocEn

4.SHOOFING EMAIL=Yaitu,tehnik menyamar sebagai instansi sebuah perusaahaan terkenal, seperti "FACEBOK,GOOGLE,dll"cara kerjanya adalah mengirimkan pesan melalui email ke email korban,dimana pesan tersebut berisi arahan untuk melakukan "verifikasi akun" dan pesan tersebut telah disisipkan dengan "link phising" atau "link malware".

Gambar4. Ilustrasi Shoofing Email

5.BRUTE FORCE ATTACK=Yaitu,serangan secara paksa untuk memasuki suatu server/web dengan cara menggunakan perintah script coding/worldlist pada tools, yang telah disiapkan untuk menjebol dan memasuki server

Gambar5. Ilustrasi Brute Force

6.DDOS ATTACK=Yaitu,Jenis serangan yang menargetkan server data base suatu website,tujuannya adalah untuk melumpuhkan akses user/pengguna ke situs tersebut dan membuat down server data base website tersebut, hingga tidak dapat di akses oleh penggunanya.Serangan ini memanfaatkan PC/Perangkat Zombie dalam melancarkan serangan DDOS Attack,dengan banyakknya PC/Perangkat zombie Yang dikendalikan oleh "penyerang" dia dapat membuat ribuan serangan ke suatu server data base website secara bersamaan,sehingga si server data base/pemberi data suatu website, menjadi error dan down, karena banyakknya permintaan data yang datang secara serentak/bersamaan,sehingga pengakses/pengguna yang sah dari suatu server website tersebut menjadi tidak dapat mengakses ke dalamnya,karena port output server data base dipenuhi dengan banyakknya permintan data base yang di buat oleh sang penyerang.

Gambar6. Ilustrasi DDOS Attack

7.SQL INJECTION=Yaitu,serangan dengan cara menyuntikkan/memasangkan suatu hardware ke server/perangkat/atau data base dengan menggunakan kabel data atau USB yang telah di program untuk menyuntikkan malware peretas sistem pada perangkat/server.

Gambar7. Ilustrasi SQL Injection
